Short Description
This project uses a Multilayer Perceptron (Neural Network) built in Python to recognize handwritten digits from the MNIST dataset. 

Folders
* src/: Contains the Python code (Jupyter Notebook).
* report/: Contains the final project PDF.
* data/: This folder is used to tell about the MNIST dataset files when the code runs.

 Instructions to Run the Notebook
1. Open Google Colab in your web browser.
2. Click File > Upload notebook, and upload the file from the `src` folder.
3. Click Runtime > Run all. The code will automatically download the dataset and train the AI.
